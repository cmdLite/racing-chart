## 🚗 Install Locally  
1️⃣ Open your terminal and zoom into your project folder. Then, clone the repo:  
   ```sh
   git clone https://github.com/cmdLite/Simple-Racing-Chart.git
   ```  
2️⃣ Enter the repo folder and install the dependencies:  
   ```sh
   npm install
   ```  
3️⃣ Start the App 
   ```sh
   npm start
   ```
